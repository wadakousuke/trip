## サイト概要
### サイトテーマ
旅行をした時に使用した駐車場、ホテル、飲食店、観光名所などを記録、評価して保存する。また、他ユーザーと共有することを目的とするSNS、レビューサイトです。

### テーマを選んだ理由
趣味や仕事でいろいろな土地に旅行する機会があるがその際に使用した施設や店舗などを記録して、評価を行い、次回以降同じ土地に旅行した際に
参考としたいと考えたため。
また、自分以外のユーザーが記録し評価した施設なども知ることができると初めていった土地でも参考にできるため。
